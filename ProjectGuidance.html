DIY Calculator

Step 1: Use components from the starter kit: Arduino R3, push buttons, and an LCD screen.

Step 2: Wire the push buttons as input, the LCD screen to display the numbers, and use resistors to limit current.

Step 3: Upload the calculator code using the Arduino IDE.

Step 4: Test by performing basic arithmetic operations (addition, subtraction, etc.).

LED Blinker

Step 1: Gather components: Arduino R3, a single LED, and a 220-ohm resistor.

Step 2: Connect the LED to a digital pin on the Arduino via the resistor (e.g., pin 13).

Step 3: Write or upload the blinking code to the Arduino IDE.

Step 4: Test by changing the delay in the code to adjust blinking frequency.

Temperature Sensor with Alert

Step 1: Gather components: Arduino R3, a DHT11 sensor, a buzzer, and jumper wires.

Step 2: Connect the DHT11 sensor to the Arduino to read temperature and humidity.

Step 3: Write a program to sound an alert via the buzzer when the temperature exceeds a set limit.

Step 4: Upload the code using Arduino IDE and monitor the readings.

Basic Obstacle Avoidance Robot

Step 1: Use components: Arduino R3, ultrasonic sensor, DC motors, motor driver, and a small chassis.

Step 2: Assemble the chassis, attach the motors, and connect the ultrasonic sensor for detecting obstacles.

Step 3: Write or upload code to drive the robot and avoid obstacles.

Step 4: Test the robot by placing it in a space with objects to navigate around.
