Project Troubleshooting

Project 1: DIY Calculator Troubleshooting

Issue: Arduino is not recognizing the input from the buttons.

Solution 1: Ensure the wiring is correct. Double-check that the push buttons are connected to the right digital pins and that they aren't floating (use pull-up or pull-down resistors).

Solution 2: Test the buttons individually using simple code to confirm they are working properly.

Issue: LCD is not displaying anything.

Solution 1: Verify the wiring between the Arduino and the LCD screen. Ensure the contrast potentiometer is adjusted to a visible level.

Solution 2: Test the LCD with a basic "Hello World" program to ensure it's functional before proceeding with your project code.

Project 2: LED Blinker Troubleshooting

Issue: LED not blinking.

Solution 1: Check the wiring. Ensure the LED is connected to the correct digital pin (e.g., pin 13) and that the resistor is placed correctly in series.

Solution 2: Use a simple blink code to test the LED. This will confirm whether the issue is with the code or the hardware.

Project 3: Temperature Sensor with Alert Troubleshooting

Issue: Sensor readings are incorrect.

Solution 1: Double-check the wiring of the DHT11 sensor to ensure it's properly connected to the Arduino.

Solution 2: Test the sensor with basic code to check if it returns valid temperature and humidity readings before proceeding with the alert code.

Project 4: Basic Obstacle Avoidance Robot Troubleshooting

Issue: Robot is not moving.

Solution 1: Check the motor wiring and motor driver connections. Ensure the power supply to the motors is sufficient.

Solution 2: Test the motors individually by providing power directly to them to confirm they're operational.

Issue: Ultrasonic sensor is not detecting obstacles.

Solution 1: Ensure the ultrasonic sensor is correctly connected to the Arduino and positioned to face obstacles.

Solution 2: Test the sensor by running a basic distance-measuring code to check its functionality.
